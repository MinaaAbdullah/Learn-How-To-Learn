Dart

- What is Dart ?

Dart is an open source, multi-purpose programming language that supports
object-oriented programming (OOP). Google developed this language and
launched the first version of it in 2011 AD, and the aim of this language
was to provide a means through which one can write one code that works on
multiple platforms Cross-Platform Language, the code written in this language
can work on the web, the Android system, the Android system ios without the need
to rewrite or make any changes to the code
The language was influenced by the way its codes were written "Syntax" in several
languages, but it is closer to the method of the C language as well as the Java
language, and the developers of the language tried to design it to replace the
JavaScript language to create applications on the client side Apps, but they
could not excel in this The point, but its compatibility with the JavaScript
language made it easy to integrate and compile within the language codes, so that
it can be used to write optimized JavaScript code compatible with web browsers
