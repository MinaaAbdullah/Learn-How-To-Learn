Swift

- What is Swift ?

The Swift language saves business execution time, unchanged files are recompiled
. Codecs in the Swift language are relatively small
. The possibility of replacing parts of the application code using the
Objective-C programming language to the Swift language
Swift has a very good memory
Swift provides easy-to-understand symbols with a neater appearance, which helps with readability
. Unlike the Objective-C programming language, Swift allows interaction with running applications

- DisAdvantages of Swift ?

The Swift programming language is a very young language, and therefore it has
not been used primarily in programming
. Swift is a very unstable language, programmers cannot use this new language to modify programs,
they have to rewrite the entire code
. Despite Apple's hard work to make Swift easy and fun, many programmers are not willing to
learn a new programming language.
. Swift is still not well suited to help programmers choose tools that help with certain tasks,
and many of its users have reported a lot of bugs and issues.
. Swift is not supported by operating systems prior to iOS7, and therefore cannot be used to develop
software for older operating systems.
